Referências:

Python

Python é uma linguagem de alto nível, embora não seja usada principalmente para desenvolvimento web, é possível usá-la. Pode ser complementar à disciplina, mesmo que o foco não seja Python. Alguns pontos de consideração incluem o uso de frameworks como o Web.py, que é full-stack, demonstrar outras linguagens disponíveis para desenvolvimento web e apresentar a linguagem para lógica de programação.

Git

Git é um sistema de controle de versão (VCS) usado para controlar as mudanças em arquivos. Pode ser usado na disciplina para hospedar projetos e avaliações, bem como para atualizar o código.

Visual Studio Code (VSCode)

VSCode é uma IDE usada para escrever e compilar código e é uma das mais completas atualmente.

Navegadores

Alguns dos navegadores que podem ser usados para visualizar páginas web incluem:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge
